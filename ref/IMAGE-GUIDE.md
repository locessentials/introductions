# Images Directory

## Required Files

### Branding
- `LocCapTransparentTilted.png` - Main logo (tilted cap)
- `LocCapLogoTransparent.png` - Alternative logo
- `favicon.png` - Browser icon

### Course Badges (50x50px recommended)
- `badge-google-classroom.png` - Globe icon for Google Classroom courses
- `badge-uic.png` - Eagle icon for UIC courses
- `badge-uwm.png` - Panther icon for UW-Milwaukee courses
- `badge-open-source.png` - Mangrove tree icon for open source courses

### Student Photos
Format: `firstname-lastname.jpg` or `.png`
- Keep file sizes under 1MB
- Recommended dimensions: 400x400px square

### Alter Ego Photos
Format: `firstname-lastname-alter-ego.jpg` or `.png`
- Keep file sizes under 1MB
- Recommended dimensions: 400x400px square

## Naming Conventions

- Use lowercase letters
- Use hyphens instead of spaces
- Include full name in filename
- Use descriptive alt text in JSON

Examples:
- `jane-doe.jpg`
- `jane-doe-alter-ego.jpg`
- `john-smith.png`
- `john-smith-alter-ego.png`

## Image Guidelines

1. **File Size**: Keep under 1MB for faster loading
2. **Dimensions**: 400x400px for square photos works well
3. **Format**: JPG for photos, PNG for graphics/logos
4. **Quality**: Balance quality and file size
5. **Accessibility**: Always provide alt text in JSON

## Creating Course Badges

You can create badges from the course cover images:

1. Open course cover in image editor
2. Extract the silhouette (eagle, panther, tree, globe)
3. Resize to 50x50px
4. Save with transparent background as PNG
5. Name according to convention above

Alternatively, use the provided course covers and let the CSS handle sizing.
