# Audio Directory

## File Format

All pronunciation files should be:
- **Format**: MP3
- **Bitrate**: 128kbps or lower (voice quality is sufficient)
- **Duration**: 1-3 seconds (just the name)
- **Size**: Keep under 100KB

## Naming Convention

Format: `firstname-lastname.mp3`

Examples:
- `jane-doe.mp3`
- `john-smith.mp3`
- `maria-garcia.mp3`

## Recording Instructions for Students

### Using Audacity (Free, Recommended)

1. Download Audacity from https://www.audacityteam.org/
2. Click the red record button
3. Say your full name clearly: "My name is [Your Name]"
4. Click stop (square button)
5. File > Export > Export as MP3
6. Name the file with your full name (e.g., jane-doe.mp3)
7. Upload to the form or send to instructor

### Using Voice Recorder (Windows)

1. Open Voice Recorder app
2. Click record
3. Say your name clearly
4. Click stop
5. Right-click recording > Open file location
6. Convert to MP3 if needed
7. Rename file to match convention

### Using Voice Memos (Mac/iOS)

1. Open Voice Memos app
2. Tap record button
3. Say your name clearly
4. Tap stop
5. Share > Save to Files
6. Convert to MP3 if needed (use online converter)
7. Rename file to match convention

## Audio Quality Tips

- **Quiet environment**: Record in a quiet room
- **Clear speech**: Speak clearly at normal pace
- **Distance**: Hold device/mic 6-12 inches from mouth
- **Volume**: Speak at normal conversational volume
- **Single take**: One clean recording of your name

## Optional: Pronunciation Guide

If your name is challenging to pronounce, you can say:
"My name is [Your Name], pronounced [phonetic spelling]"

Example: "My name is Alaina, pronounced ah-LAY-nah"

## Troubleshooting

### File too large
- Use lower bitrate (96kbps or 64kbps is fine for voice)
- Trim silence from beginning and end
- Keep recording under 3 seconds

### Wrong format
- Use online converter (e.g., CloudConvert.com)
- Or re-export from Audacity as MP3

### Can't hear playback
- Check volume settings in browser
- Verify file uploaded correctly
- Try different browser

## Privacy Note

These audio files will be:
- Shared with classmates during the course
- Included on public profile if you consent
- Used to help others pronounce your name correctly

If you prefer not to include audio, leave the field blank on the submission form.
